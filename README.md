![Preview](preview/docker-nginx-express.gif)

# Development

* Cloning the repo

```bash
$ git clone [Repo URL]
```

* Installing dependencies

```bash
$ npm install
```

# Docker

* Building an image

```bash
$ docker-compose build
```

* Running containers

```bash
$ docker-compose up -d
```

* Stopping  containers

```bash
$ docker-compose down
```
